# Luan Junqueira | Portfolio

Este repositório contém o código-fonte do meu portfólio profissional, focado em Desenvolvimento Full Stack, Automação Inteligente e IA.

## Tecnologias Utilizadas

- **Vite** & **React**
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui**
- **Lucide React**

## Como Executar Localmente

Siga os passos abaixo para rodar o projeto em sua máquina:

1. **Clone o repositório:**
   ```sh
   git clone https://github.com/luanjunqueira/LuanPortifolioOpenAutomate.git
   ```

2. **Acesse a pasta do projeto:**
   ```sh
   cd LuanPortifolioOpenAutomate
   ```

3. **Instale as dependências:**
   ```sh
   npm install
   ```

4. **Inicie o servidor de desenvolvimento:**
   ```sh
   npm run dev
   ```
